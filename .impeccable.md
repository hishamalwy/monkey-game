## Design Context

### Users
Arabic-speaking mobile users playing casually or competitively. Whether in local offline mode (passing the device) or in real-time online rooms. They need fast, highly responsive inputs and clear feedback.

### Brand Personality
1. Energetic
2. Quirky
3. Pressuring/Fast-paced

### Aesthetic Direction
A bold maximalist arcade vibe. Vibrant hues, thick outlines, chunky interactive elements. It should feel like a colorful, high-energy mobile game. Emphasizes tactile satisfaction (button presses, animations, clear contrast).

### Design Principles
1. **Bold Contrast**: Ensure legibility under pressure. Use the existing bright yellow / deep purple contrast for core typography.
2. **Tactile Delight**: Every interaction (like typing a letter or passing a turn) must have a visual and kinetic response.
3. **Fluid Layout**: While it's a mobile-first game, it shouldn't awkwardly stretch on web desktop platforms. It needs constrained central game boards with responsive scaling.
