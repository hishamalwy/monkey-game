## Buzzer Behavior Correction (Critical Fix)

The buzzer must follow this exact logic:

* The buzzer is ALWAYS available (open) by default

* When ANY player presses:
  → the FIRST player wins
  → all other players are instantly locked
  → buzzer becomes LOCKED

* While a player is answering:
  → no one else can press

* After the host makes a decision:

  * if answer is CORRECT:
    → round ends
    → buzzer resets to OPEN for next round

  * if answer is WRONG:
    → apply minus / penalty
    → buzzer immediately returns to OPEN
    → remaining players can press again

IMPORTANT:

* The buzzer MUST NOT reopen automatically
* It must reopen ONLY after the host decision

This logic is mandatory and must override any previous implementation.
